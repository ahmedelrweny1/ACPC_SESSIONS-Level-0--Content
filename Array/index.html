What is an Array?

An array is a collection of elements of the same data type stored in contiguous memory locations. It's one of the most fundamental data structures in programming.

Key Characteristics

  • Fixed Size: Size is determined at compile-time or runtime
  • Homogeneous: All elements are of the same type
  • Indexed: Elements are accessed using indices (0-based)
  • Contiguous: Elements are stored sequentially in memory
C++ 
#include <iostream>
using namespace std;

int main() {
    // Simple array declaration
    int numbers[5] = {10, 20, 30, 40, 50};
    
    // Display array elements
    cout << "Array elements: ";
    for(int i = 0; i < 5; i++) {
        cout << numbers[i] << " ";
    }
    cout << endl;
    
    return 0;
}

// Output: Array elements: 10 20 30 40 50

Memory Representation & Runtime vs Compile Time

üïê Compile Time

When the compiler processes your code and determines array size before the program runs.

int arr[5];  // Size known at compile time
const int SIZE = 10;
int data[SIZE];  // Also compile time

‚öôÔ∏è Runtime

When the program is executing and array size can be determined dynamically.

int n;
cin >> n;
int arr[n];  // Size determined at runtime
             // (VLA - not standard C++)
‚ö†Ô∏è Note: Variable Length Arrays (VLA) are not part of standard C++, though some compilers support them as an extension. For truly dynamic arrays, use std::vector or dynamic memory allocation.

Declaration & Initialization

Declaration vs Initialization

Declaration: Reserves memory space for the array

Initialization: Assigns values to array elements

Initialization Methods

1

Declaration Only (with default values)

int arr[5];  // Uninitialized - contains garbage values
int arr2[5] = {};  // All elements initialized to 0
int arr3[5] = {0};  // All elements initialized to 0
‚ö†Ô∏è Warning: Uninitialized arrays hold garbage values - unpredictable data from previous memory usage!
2

Initialize Immediately

int arr[5] = {10, 20, 30, 40, 50};  // Full initialization
int arr2[5] = {1, 2};  // Partial: {1, 2, 0, 0, 0}
double grades[4] = {95.5, 87.3, 92.1, 88.8};
3

Without Specifying Size

int arr[] = {1, 2, 3, 4, 5};  // Size automatically = 5
char vowels[] = {'a', 'e', 'i', 'o', 'u'};  // Size = 5
// Compiler determines size from initializer list
4

Using External Size (Runtime)

#include <iostream>
#include <vector>  // Better alternative
using namespace std;

int main() {
    int n;
    cout << "Enter size: ";
    cin >> n;
    
    // Using vector (recommended)
    vector<int> arr(n);
    
    // Or dynamic allocation
    int* dynamicArr = new int[n];
    // ... use array ...
    delete[] dynamicArr;  // Don't forget!
    
    return 0;
}
‚ö†Ô∏è Segmentation Fault: Accessing beyond array bounds causes undefined behavior and crashes!

Array Indexing

Understanding Indices

  • Arrays use 0-based indexing
  • First element is at index 0
  • Last element is at index (size - 1)
  • Valid range: [0, size-1]

Memory Address Formula

Address of arr[i] = base_address + (i √ó element_size)

#include <iostream>
using namespace std;

int main() {
    int numbers[5] = {100, 200, 300, 400, 500};
    
    // Accessing elements
    cout << "First element: " << numbers[0] << endl;
    cout << "Third element: " << numbers[2] << endl;
    cout << "Last element: " << numbers[4] << endl;
    
    // Modifying elements
    numbers[1] = 250;
    cout << "Modified second: " << numbers[1] << endl;
    
    // Using indices in expressions
    int sum = numbers[0] + numbers[4];
    cout << "Sum of first and last: " << sum << endl;
    
    return 0;
}

/* Output:
First element: 100
Third element: 300
Last element: 500
Modified second: 250
Sum of first and last: 600
*/

Built-in Array Functions

C++ provides powerful STL algorithms to work with arrays efficiently. Include <algorithm> and <numeric> headers.

üîÄ sort()

Sorts array elements in ascending or descending order

#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int arr[] = {45, 12, 78, 23, 67};
    int n = 5;
    
    // Ascending sort using begin/end
    sort(begin(arr), end(arr));
    
    cout << "Ascending: ";
    for(int x : arr) cout << x << " ";
    cout << endl;
    // Output: 12 23 45 67 78
    
    // Descending sort using greater<>
    sort(arr, arr + n, greater<int>());
    
    cout << "Descending: ";
    for(int x : arr) cout << x << " ";
    cout << endl;
    // Output: 78 67 45 23 12
    
    return 0;
}

üìä max_element() & min_element()

Find maximum and minimum elements

#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int arr[] = {45, 12, 78, 23, 67};
    
    // Using iterators
    int maxVal = *max_element(begin(arr), end(arr));
    int minVal = *min_element(begin(arr), end(arr));
    
    cout << "Max: " << maxVal << endl;  // 78
    cout << "Min: " << minVal << endl;  // 12
    
    // Alternative: Using pointers
    int* maxPtr = max_element(arr, arr + 5);
    cout << "Max at index: " 
         << (maxPtr - arr) << endl;  // 2
    
    return 0;
}

2D Arrays (Matrices)

A 2D array is an array of arrays - think of it as a table with rows and columns.

Syntax

datatype arrayName[rows][columns];

Initialization

#include <iostream>
using namespace std;

int main() {
    // Method 1: Full initialization
    int matrix1[3][3] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    
    // Method 2: Linear initialization
    int matrix2[2][3] = {1, 2, 3, 4, 5, 6};
    
    // Method 3: Partial initialization
    int matrix3[3][3] = {
        {1, 2},     // Rest are 0: {1, 2, 0}
        {3},        // {3, 0, 0}
                    // {0, 0, 0}
    };
    
    return 0;
}

Accessing Elements

#include <iostream>
using namespace std;

int main() {
    int matrix[3][3] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    
    // Access individual element
    cout << "Element at [1][2]: " 
         << matrix[1][2] << endl;  // 6
    
    // Modify element
    matrix[0][0] = 10;
    
    return 0;
}

Input & Output

#include <iostream>
using namespace std;

int main() {
    int rows = 3, cols = 3;
    int matrix[3][3];
    
    // Input
    cout << "Enter 3x3 matrix:" << endl;
    for(int i = 0; i < rows; i++) {
        for(int j = 0; j < cols; j++) {
            cin >> matrix[i][j];
        }
    }
    
    // Output
    cout << "\nMatrix:" << endl;
    for(int i = 0; i < rows; i++) {
        for(int j = 0; j < cols; j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }
    
    return 0;
}

Frequency Arrays

What is a Frequency Array?

A frequency array is a special technique where:

  • Index = Value from the original array
  • Value = Count (how many times it appears)

It's perfect for counting occurrences of integers within a known range!

‚úÖ Use Cases

  • Count occurrences of elements
  • Find mode (most frequent value)
  • Check if number appeared before
  • Sort integers in linear time

‚ö° Time Complexity

O(n) - Linear time

Much faster than nested loops O(n¬≤)

Basic Example

#include <iostream>
using namespace std;

int main() {
    int arr[] = {1, 3, 2, 1, 4, 3, 1, 2};
    int n = 8;
    
    // Frequency array (assuming values 0-100)
    int freq[101] = {0};  // Initialize all to 0
    
    // Count frequencies
    for(int i = 0; i < n; i++) {
        freq[arr[i]]++;
    }
    
    // Display frequencies
    cout << "Frequencies:" << endl;
    for(int i = 0; i <= 10; i++) {
        if(freq[i] > 0) {
            cout << i << " appears " 
                 << freq[i] << " times" << endl;
        }
    }
    
    return 0;
}

/* Output:
Frequencies:
1 appears 3 times
2 appears 2 times
3 appears 2 times
4 appears 1 times
*/

Find Mode (Most Frequent)

#include <iostream>
using namespace std;

int main() {
    int arr[] = {1, 3, 2, 1, 4, 3, 1, 2};
    int n = 8;
    int freq[101] = {0};
    
    // Count frequencies
    for(int i = 0; i < n; i++) {
        freq[arr[i]]++;
    }
    
    // Find mode
    int mode = 0, maxFreq = 0;
    for(int i = 0; i <= 100; i++) {
        if(freq[i] > maxFreq) {
            maxFreq = freq[i];
            mode = i;
        }
    }
    
    cout << "Mode: " << mode 
         << " (appears " << maxFreq 
         << " times)" << endl;
    // Output: Mode: 1 (appears 3 times)
    
    return 0;
}

Check if Number Appeared

#include <iostream>
using namespace std;

int main() {
    int arr[] = {5, 10, 15, 20, 25};
    int n = 5;
    int freq[101] = {0};
    
    // Mark appearances
    for(int i = 0; i < n; i++) {
        freq[arr[i]] = 1;  // Just mark as seen
    }
    
    // Check queries
    int query = 15;
    if(freq[query]) {
        cout << query << " is present" << endl;
    } else {
        cout << query << " is not present" << endl;
    }
    
    return 0;
}

Sort Using Frequency (Counting Sort)

#include <iostream>
using namespace std;

int main() {
    int arr[] = {4, 2, 2, 8, 3, 3, 1};
    int n = 7;
    int freq[101] = {0};
    
    // Count frequencies
    for(int i = 0; i < n; i++) {
        freq[arr[i]]++;
    }
    
    // Print sorted array
    cout << "Sorted array: ";
    for(int i = 0; i <= 100; i++) {
        for(int j = 0; j < freq[i]; j++) {
            cout << i << " ";
        }
    }
    cout << endl;
    // Output: Sorted array: 1 2 2 3 3 4 8
    
    return 0;
}
